# Real-time chat on React

#### `react.js + firebase`

## Локальный запуск

1. Выполнить команду npm `npm i`
2. Создать файл `.env` в корне проекта
3. Заполнить файл `.env` соответсвующими ключами ваше `firebase` проекта
4. Выполнить команду `npm start`
